# HashMap

A custom HashMap implementation built with JavaScript for The Odin Project.

## Features

- Create a HashMap class
- Store key-value pairs
- Handle collisions
- Retrieve values by key
- Check if a key exists
- Remove entries
- Return all keys, values, and entries
- Clear the hashmap
- Automatically resize when the load factor is reached

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Test

Open `index.html` in the browser, then open the developer console.
